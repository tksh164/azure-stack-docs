# Get-AzSDnsForwarder

## Synopsis

Gets the DNS forwarder configuration.

## Syntax

```
Get-AzSDnsForwarder
```

## Description

Gets the list of DNS forwarder IP addresses. You can update the configuration by using the [Set-AzSDnsForwarder](set-azsdnsforwarder.md) cmdlet.

## Examples

### Example 1

```
Get-AzSDnsForwarder
```

## Parameters

None.
