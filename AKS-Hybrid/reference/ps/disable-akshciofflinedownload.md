# Disable-AksHciOfflinedownload

## Synopsis

Disables offline downloading to get the AKS hybrid images.

## Syntax

```powershell
Disable-AksHciOfflinedownload  -stagingShare
```

## Description

Disables offline downloading to get the AKS hybrid images.

## Examples

```PowerShell
Disable-AksHciOfflinedownload -stagingShare
```
